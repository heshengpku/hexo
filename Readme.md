# Hexo

## Installation
```
$ npm install hexo-cli -g
```

## Init
```
$ hexo init blog
$ cd blog
```

## Install pandoc for latex formula
https://pandoc.org/installing.html

## Start the server
```
$ hexo server
```

## Create a new post
```
$ hexo new "Hello Hexo"
```

## Generate static files
```
$ hexo generate
```

## Deploy
```
$ hexo deploy
```
